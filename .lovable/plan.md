## üî≤ Panel Grid View

Three visualization modes accessible via tabs:

**Physical Layout (Aerial View):**
- Bird's-eye visualization of panel rows
- Color-coded status (healthy/warning/fault)
- Zoom and pan functionality
- Click panel to see details

**Logical/Electrical Diagram:**
- Schematic showing string connections
- Inverter groupings visualized

**Table View:**
- Sortable columns (ID, status, efficiency, last checked)

- Bulk selection for actions
- Export capability
